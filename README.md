Svgcleaner docker image (minidocks/svgcleaner)
==============================================

svgcleaner helps you clean up your SVG files, keeping them free from unnecessary data.

Usage
-----

``` bash
docker run --rm -v `pwd`:/app -w /app minidocks/svgcleaner --help
```
